# Portfolio

This project is my portfolio. It allows me to introduce myself to you, showcase my projects, and provide a way to contact me.

## Prerequisites

- Node.js 18.17
- Npm
- Email account (e.g., gmail) - required for email sending features.

## Getting Started

To start working on this project, follow these simple steps:

### Installation

1. Clone this repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the application in development mode.

### Configuration

1. Create a `.env` file at the root of the project.
2. Add and fill in the following environment variables:

> EMAIL_USER=your email address
> EMAIL_PASSWORD=application password
> EMAIL_PORT=mail server port
> EMAIL_HOST=mail server host
> EMAIL_SECURE=mail server security protocol

Open [http://localhost:3000](http://localhost:3000/) with your browser to see the result.
